Sobre mim

Olá, me chamo Guilherme Jorge de Araujo! Sou estudante de Desenvolvimento de Sistemas na escola SENAI, e já deslumbrei alguns dos campos correlatos a área da Tecnologia Informática, como HTML, CSS, JS e Banco de Dados.

Meu Projeto

Este projeto consiste em um portfólio Web desenvolvido com o objetivo de apresentar os conceitos que aprendi, e evolução na área de Desenvolvimento de Sistemas. A aplicação foi construída utilizando HTML e CSS. O projeto busca aplicar conceitos fundamentais de Desenvolvimento Front-end, como organização estrutural, estilização hodierna e experiência do usuário. Objetivos do projeto: Consolidar conhecimentos em Desenvolvimento Web; Desenvolver uma interface voltada à apresentação profissional; Servir como vitrine para projetos e habilidades em desenvolvimento.
